Generating identicon image!

![kaneshin.png](https://github.com/kaneshin/identicon/raw/master/kaneshin.png)

## Installation

```shell
$ go get -u github.com/kaneshin/identicon/cmd/identicon
```

## Usage

```shell
# To generate kaneshin.png
$ identicon kaneshin
```
